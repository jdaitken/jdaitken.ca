## Social Avatar / Preview

Use the geometric diamond symbol only (no text) on a solid background with ample padding.

Recommended export:
- Square 512×512 (or 1024×1024) PNG and WebP.
- Background: use `#0B1220` (charcoal) or `#131B2B` with the `diamond-mark--light` mark.
- Keep flat colors; no gradients or shadows.

File naming: `diamond-mark--social-512.png` (and WebP if desired).
